สรุปรายการงานที่ดำเนินการล่าสุด (2026-03-23)

1. การปรับปรุงโครงสร้างข้อมูลสำหรับ "ระบบลูกข่าย" (Client System):
   - เพิ่มคอลัมน์ใหม่ในแผ่นงาน "ClientSystem" ทั้งหมด 4 คอลัมน์ ได้แก่ ขาติดตั้ง, องศา, ค่า Test Feeder, ยื่นขอมิเตอร์

2. การพัฒนา API (WebApp):
   - ปรับปรุงไฟล์ `src/app/api/sheet-data/route.ts`:
     * เพิ่มการ Mapping ข้อมูลจากคอลัมน์ใหม่ (W, X, Y, Z) เข้าสู่ระบบสำหรับ Client System
     * ขยายระยะการอ่านและเขียนข้อมูล (Range) จาก A:W เป็น A:Z
     * รองรับฟิลด์ข้อมูลใหม่ในการทำ POST และ PUT

3. การพัฒนาส่วนติดต่อผู้ใช้งาน (UI) และการเพิ่มเพิ่มข้อมูลหน้าเว็บ:
   - ปรับปรุงไฟล์ `src/components/ClientSystemModal.tsx`:
     * เพิ่มฟิลด์กรอกข้อมูลรูปแบบ Drop down สำหรับ "ขาติดตั้ง" (เช่น A, B, C, D)
     * เพิ่มฟิลด์สำหรับกรอก "องศา" (เป็นตัวเลข)
     * เพิ่มฟิลด์ตั้งสถานะ "ค่า Test Feeder" (ยังไม่ได้เก็บ, เก็บแล้ว)
     * เพิ่มฟิลด์ตั้งสถานะ "ยื่นขอมิเตอร์" (ยังไม่ได้ยื่น, รออนุมัติ, ติดตั้งแล้ว)

4. การปรับปรุงระบบ Export รายงาน .txt:
   - ปรับปรุงไฟล์ `src/app/page.tsx`:
     * แก้ไข Format ของรายงานสถานะระบบลูกข่ายเป็นแบบใหม่ตามที่ระบุ
     * แสดงข้อมูลใหม่ทั้ง 4 ค่าลงในไฟล์ Text สรุปงานได้อย่างครบถ้วน

สถานะปัจจุบัน:
- หน้าเว็บพร้อมใช้งานร่วมกับ Google Sheet รูปแบบใหม่แล้ว
- รายงาน .txt (สรุปงาน) จะได้หน้าตาข้อมูลคอลัมน์ใหม่ที่เพิ่มเข้ามาใช้งานเรียบร้อย
