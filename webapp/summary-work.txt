สรุปรายการงานที่ดำเนินการล่าสุด (2026-03-19)

1. การพัฒนาโครงสร้างข้อมูล (Google Sheets):
   - เพิ่มคอลัมน์ใหม่ในแผ่นงาน "ClientSystem" เพื่อรองรับข้อมูลอุปกรณ์วิทยุ:
     * SN เครื่องวิทยุ MT680 Plus (คอลัมน์ R)
     * SN แบตเตอรี่ 50AH (คอลัมน์ S)
     * ค่า RSSI dBm (คอลัมน์ T)
   - ปรับปรุงสคริปต์ `scripts/update-headers.ts` ให้รองรับการอัปเดตหัวตาราง (Headers) ตามโครงสร้างใหม่

2. การพัฒนา API (WebApp):
   - ปรับปรุงไฟล์ `src/app/api/sheet-data/route.ts`:
     * เพิ่มการ Mapping ข้อมูลจากคอลัมน์ใหม่ (R, S, T) เข้าสู่ระบบ
     * ขยับคอลัมน์ Remark, StartDate และ EndDate ไปยังตำแหน่งใหม่ (U, V, W)
     * รองรับการบันทึกข้อมูล (POST/PUT) สำหรับฟิลด์ใหม่ทั้งหมด

3. การพัฒนาส่วนติดต่อผู้ใช้งาน (UI):
   - ปรับปรุงไฟล์ `src/components/ClientSystemModal.tsx`:
     * เพิ่มฟิลด์กรอกข้อมูล Serial Number สำหรับเครื่องวิทยุและแบตเตอรี่
     * เพิ่มช่องสำหรับกรอกค่า RSSI (dBm)
     * ปรับปรุง Dynamic Layout ให้แสดงผลสวยงามและรองรับโหมด Dark Mode

4. การดำเนินการระบบ (Deployment):
   - อัปเดตหัวตารางใน Google Sheet จริงเรียบร้อยแล้วผ่านสคริปต์
   - ดำเนินการ Commit และ Push โค้ดทั้งหมดขึ้น GitHub (Repository: bearnannan/prossnum)

สถานะปัจจุบัน:
- ระบบพร้อมรองรับการกรอกข้อมูลใหม่แล้ว
- *หมายเหตุ: ยังไม่มีการเพิ่มข้อมูลใหม่ (Data entries) เข้าไปในระบบในรอบนี้ รอข้อมูลจากผู้ใช้งาน*
